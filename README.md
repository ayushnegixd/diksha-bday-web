# 🎂 Personalized Birthday Scrapbook

A high-end, mobile-optimized digital scrapbook built for a friend. This project focuses on **Glassmorphism UI/UX**, responsive layouts, and interactive frontend elements.

---

## ✨ Features

* **Glassmorphism UI**: High-end frosted glass cards for all 10 memory modules, creating a premium acrylic feel.
* **Mobile-First Design**: Perfectly centered 2-column grid layout optimized for iPhone and Android viewports.
* **Secure Entrance**: A custom lock screen requiring a DDMM-format password to access the memories.
* **Interactive Celebrations**: Automatic music playback and confetti triggers upon successful unlock.
* **Dynamic Typography**: Custom pink-to-red gradients paired with elegant serif and handwritten fonts for a professional visual hierarchy.

---

## 🚀 Tech Stack

* **Frontend**: React.js with Vite.
* **Styling**: Tailwind CSS (Custom configurations for Glassmorphism and Gradients).
* **Animations**: Framer Motion for smooth scroll reveals and transitions.
* **Effects**: Canvas-confetti for the birthday celebration trigger.
